# refrigerator-law-simulation
A “refrigerator law” simulation typically demonstrates the second law of thermodynamics applied to refrigeration—heat flows from a cold region to a hot region using external work.
🧊 Refrigerator Law Simulation (Python - Pygame)

📌 Overview

This project is a visual simulation of the refrigerator working principle, based on the Second Law of Thermodynamics. It demonstrates how heat is transferred from a cold region to a hot region using external work.

The simulation uses Pygame to display:

- Cold region (inside refrigerator)
- Hot region (outside environment)
- Compressor (work input)
- Heat flow direction


🎯 Objectives

- Understand how refrigerators work
- Visualize heat transfer from cold to hot
- Demonstrate the role of external work (compressor)
- Provide an interactive learning tool for physics students


🧠 Concept

A refrigerator works on the principle:

Q_hot = Q_cold + Work

Where:

- Q_cold → Heat absorbed from inside the fridge
- Work → Energy supplied by compressor
- Q_hot → Heat released to surroundings

This follows the Second Law of Thermodynamics.


⚙️ Features

- Real-time temperature change simulation
- Visual representation of heat flow
- Compressor work indication
- Smooth animation using Pygame
- Beginner-friendly code


🛠️ Technologies Used

- Python 3.x
- Pygame library


📦 Installation

Step 1: Clone Repository

git clone https://github.com/your-username/refrigerator-simulation.git
cd refrigerator-simulation

Step 2: Install Dependencies

pip install pygame

Step 3: Run the Project

python main.py


▶️ How It Works

- The blue box represents the cold region (inside fridge)
- The red box represents the hot region (outside)
- Heat is absorbed from the cold region
- The compressor adds work
- Heat is released into the hot region

Temperatures update continuously to simulate cooling and heating.
